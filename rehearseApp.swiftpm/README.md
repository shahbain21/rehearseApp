# Rehearse

A SwiftUI iOS app for practicing and improving speaking skills through audio recording and feedback.

## About

Rehearse helps users prepare for presentations, interviews, and public speaking by recording practice sessions, analyzing speaking patterns, and providing actionable feedback.

## Features

- **Practice Modes** - Interview, Presentation, Storytelling, and Free Practice
- **Audio Recording** - Real-time waveform visualization with speech detection
- **Speech Analysis** - Speaking ratio, pause detection, segment length, and fluency metrics
- **Feedback System** - Overall score, detailed metrics, insights, and practice exercises
- **Grounding Exercise** - Breathing animation before recording to reduce anxiety
- **Reflection Prompts** - Guided post-session reflection
- **History** - View and playback past recordings

## Requirements

- iOS 16.0+
- Xcode 15.0+
- Microphone permission

## Installation

1. Clone the repository
2. Open `rehearseApp.xcodeproj` in Xcode
3. Build and run on simulator or device
4. Grant microphone permission when prompted

## Usage

1. Select a practice mode
2. Add notes (optional)
3. Complete the breathing exercise
4. Record your practice session
5. Review feedback and metrics
6. Reflect on your session
